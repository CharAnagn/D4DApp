# D4D Backend part

## Developer installation

### Install packages

Run `npm install` to install neccessary packages:

* body-parser
* cors
* express
* app-root-path
* file-exists
* node-env-file

### Configuration

1. Copy *.env.examle* to *.env.local*
2. Change parameters in *.env.local* if needed for your local installation


### Run

Run `node app.js`
